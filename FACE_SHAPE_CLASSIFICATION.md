# 顔型分類システム - 5つの顔型タイプ

## 1️⃣ 卵型（たまごがた / Oval Face）

### 特徴
- **理想的な顔型**として知られる
- 顔の縦幅が横幅の約1.3〜1.5倍
- なだらかな曲線を描く輪郭
- 額から顎にかけて緩やかに細くなる

### 判定基準
```
幅/高さ比率: 0.68〜0.78
顎/額比率: 0.82〜0.95
```

### 芸能人例
- 石原さとみ
- 新垣結衣
- 北川景子

### イメージ
```
    ___
   /   \
  |     |
  |     |
   \___/
```

---

## 2️⃣ 丸顔（まるがお / Round Face）

### 特徴
- **可愛らしい印象**を与える顔型
- 縦幅と横幅がほぼ同じ
- 頬がふっくらとしている
- 顎のラインが丸みを帯びている

### 判定基準
```
幅/高さ比率: 0.82〜1.05
顎/額比率: 0.92〜1.08
```

### 芸能人例
- 有村架純
- 森七菜
- 永野芽郁

### イメージ
```
   ___
  (   )
  |   |
  (___) 
```

---

## 3️⃣ 四角顔（しかくがお / Square Face）

### 特徴
- **意志が強そうな印象**を与える顔型
- エラが張っている
- 額の幅と顎の幅がほぼ同じ
- 直線的な輪郭ライン

### 判定基準
```
幅/高さ比率: 0.78〜0.98
顎/額比率: 0.95〜1.12
エラの角度を検出
```

### 芸能人例
- 綾瀬はるか
- 天海祐希
- 深田恭子

### イメージ
```
  ┌─┐
  │ │
  │ │
  └─┘
```

---

## 4️⃣ ハート型（逆三角形 / Heart Face）

### 特徴
- **知的で上品な印象**を与える顔型
- 額が広い
- 顎が細くシャープ
- 頬骨が目立つことがある

### 判定基準
```
幅/高さ比率: 0.72〜0.88
顎/額比率: 0.55〜0.78（顎が細い）
```

### 芸能人例
- 広瀬すず
- 本田翼
- 桐谷美玲

### イメージ
```
  ╱─╲
 │   │
 │   │
  ╲V╱
```

---

## 5️⃣ 面長（おもなが / Long Face）

### 特徴
- **大人っぽい印象**を与える顔型
- 顔の縦幅が横幅より明らかに長い
- すらりとした印象
- 日本人女性に多い顔型

### 判定基準
```
幅/高さ比率: 0.52〜0.68
顎/額比率: 0.80〜0.95
```

### 芸能人例
- 水川あさみ
- 菅野美穂
- 天海祐希

### イメージ
```
  ┌┐
  ││
  ││
  ││
  └┘
```

## 📊 顔型判定のアルゴリズム

### 測定ポイント（23箇所）

1. **横幅の測定点**
   - 左右の頬骨（最も広い部分）
   - 左右のこめかみ
   - 左右のエラ

2. **縦幅の測定点**
   - 額の生え際中央
   - 眉間
   - 鼻先
   - 顎先

3. **補助測定点**
   - 左右の目尻
   - 左右の口角
   - 鼻の付け根
