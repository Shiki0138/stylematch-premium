# 💰 StyleMatch ランニングコスト分析

## 📊 月間コスト概算（ユーザー数別）

### 🎯 想定条件
- 1ユーザーあたり月間アクティブ率: 60%
- 1ユーザーあたり月間診断回数: 2回
- 1ユーザーあたり月間ページビュー: 50PV

## 💵 コスト内訳

### 1. Firebase（メインインフラ）

#### 100ユーザー規模（スタートアップ）
- **Firestore**: 無料枠内
  - 読み取り: 50,000回/日まで無料
  - 書き込み: 20,000回/日まで無料
- **Authentication**: 無料（50,000MAU まで）
- **Storage**: 無料（5GBまで）
- **Hosting**: 無料（10GB/月まで）
- **月額**: **$0**

#### 1,000ユーザー規模
- **Firestore**: $20-30
  - 読み取り: ~300,000回/月
  - 書き込み: ~100,000回/月
- **Authentication**: 無料
- **Storage**: $5（画像保存 ~25GB）
- **Hosting**: 無料
- **月額**: **$25-35**

#### 10,000ユーザー規模
- **Firestore**: $200-300
  - 読み取り: ~3,000,000回/月
  - 書き込み: ~1,000,000回/月
- **Authentication**: 無料
- **Storage**: $50（画像保存 ~250GB）
- **Hosting**: $20
- **月額**: **$270-370**

### 2. AI処理（Flask + Python）

#### Vercel Functions（推奨）
- **Hobby**: $0（月100GB-hrs まで）
- **Pro**: $20/月（月1,000GB-hrs）
- **診断1回あたり**: ~2秒の処理時間

#### 代替: Cloud Run（Google Cloud）
- **100ユーザー**: $5-10
- **1,000ユーザー**: $50-80
- **10,000ユーザー**: $500-800

### 3. Sentry（モニタリング）

- **Developer**: $0（5,000エラー/月）
- **Team**: $26/月（50,000エラー/月）
- **Business**: $80/月（100,000エラー/月）

### 4. その他のサービス

#### 必須
- **ドメイン**: $15/年 = $1.25/月
- **SSL証明書**: 無料（Let's Encrypt）

#### オプション
- **SendGrid（メール）**: $0-20/月
- **Cloudflare（CDN）**: $0-20/月
- **UptimeRobot（監視）**: $0-7/月

## 💡 コスト最適化のポイント

### 1. キャッシュ戦略
- React Queryの積極活用で読み取り回数を50%削減
- 画像のCDNキャッシュで転送量削減

### 2. 画像最適化
- クライアントサイド圧縮で保存容量を60-80%削減
- WebP形式の採用でさらに30%削減

### 3. データベース設計
- 非正規化によるクエリ削減
- バッチ処理による書き込み最適化

### 4. AI処理の効率化
- モデルの軽量化（精度を保ちつつ）
- エッジでの前処理による負荷軽減

1. **初期段階はほぼ無料**で運用可能（月額$1.25）
